Open with Visual Studio Code

28 Jul 2017, 11:53

On Windows it is possible to select a file or folder and from the context menu open this in Visual Studio Code. For macOS we need to add this functionality ourselves.

However, it is very simple using the following steps:

  1. Start Automator
  2. Create a new Service
  3. Change “Service Receives” to “files or folders” in “Finder”
  4. Add a “Run Shell Script” action
  5. Change “Pass input” to “as arguments”
  6. Paste the following in the shell script box: open -n -b "com.microsoft.VSCode" --args "$*"
  7. Save it as something like “Open in Visual Studio Code”

And that’s it.
